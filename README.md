# WhatsApp Group Tracker

⚠️ **تحذير: هذا المشروع لأغراض أمنية وقانونية فقط**

## المميزات
- صفحة مجموعة واتساب مزيفة مقنعة
- جمع معلومات IP والموقع التقريبي
- محاولة الحصول على الإحداثيات الدقيقة عبر GPS
- لوحة تحكم لعرض البيانات المجمعة
- قاعدة بيانات SQLite للتخزين

## التثبيت

```bash
# نسخ المشروع
git clone https://github.com/yourusername/whatsapp-tracker.git
cd whatsapp-tracker

# تثبيت المتطلبات
pip install -r requirements.txt

# تشغيل التطبيق
python app.py
